# Component Catalog

- `ProfileCabinetCard`: карточка пользователя для `client`, `specialist`, `manager`.
- `RoleCabinetHomePage`: role-aware home page with dashboard metrics from backend.
- `RoleProfileEditorPage`: shared role profile form with local + backend persistence.
- `input`: single-line user input.
- `textarea`: multi-line notes.
- `phone_input`: phone-formatted input.
- `email_input`: email-formatted input.
- `button`: primary call-to-action.

All generated components bind to typed variables from the AppIR, while role-specific dashboards come from backend seed data.
